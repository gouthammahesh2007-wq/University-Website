# Adi Shankara Institute of Engineering and Technology  Website

## 📌 Description

This is a responsive university website built using HTML, CSS, and JavaScript.
It features a modern design with a homepage, image slider, course details, and an informative About section.

---

## 🚀 Features

* Responsive design (mobile-friendly)
* Smooth navigation
* Image slider for campus view
* About section with detailed information
* Courses offered section
* Contact details footer

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

---

## 📂 Project Structure

```
project-folder/
│
├── index.html
├── style.css
└── script.js
```

---

## ▶️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Double-click on `index.html`
4. View in your browser

A simple and clean university website interface with navigation, campus images, and course details.
