Adi Shankara Institute of Engineering and Technology Website
📌 Description
This is a responsive university website built using HTML, CSS, and JavaScript. It features a modern design with a homepage, image slider, course details, and an informative About section.

🚀 Features
Responsive design (mobile-friendly)
Smooth navigation
Image slider for campus view
About section with detailed information
Courses offered section
Contact details footer
🛠️ Technologies Used
HTML5
CSS3
JavaScript

project-folder/

├── index.html
├── style.css

▶️ How to Run
Download or clone the repository
Open the project folder
Double-click on index.html
View in your browser
